# File Lancet v0.1 Release Notes

## 版本信息
- **版本**: v0.1
- **发布日期**: 2026-03-26
- **支持平台**: Windows 10/11 (x64)
- **运行时**: .NET 10 (self-contained)

## 下载

### 便携版 (Portable)
- **文件名**: FileLancet-v0.1-win-x64.zip
- **说明**: 解压后直接运行 FileLancet.UI.exe，无需安装 .NET 运行时

## 功能特性

### 核心功能
- ✅ EPUB 2.0/3.0/3.1 文件解析
- ✅ 文件结构树形浏览
- ✅ 元数据提取（书名、作者、出版社、语言等）
- ✅ 内容预览（HTML、CSS、JavaScript、XML、图片）
- ✅ 十六进制预览（二进制文件）

### 界面特性
- ✅ 双栏式布局（文件树+详情 / 预览）
- ✅ 拖拽打开文件
- ✅ 文件关联支持
- ✅ 自适应调整面板大小

### 技术特性
- ✅ 懒加载优化
- ✅ 内容缓存
- ✅ 异步处理
- ✅ 扩展架构（支持添加新文件格式）

## 系统要求
- Windows 10/11 (x64)
- 4GB 内存（推荐）
- 100MB 磁盘空间

## 使用方法
1. 下载并解压 FileLancet-v0.1-win-x64.zip
2. 运行 FileLancet.UI.exe
3. 点击 "Open" 按钮或拖拽 EPUB 文件到窗口

## 已知问题
- 当前版本仅支持 EPUB 格式
- 大文件（>100MB）加载可能需要较长时间
- 某些非标准 EPUB 文件可能解析失败

## 未来计划
- 支持更多文件格式（PDF、MOBI 等）
- 添加搜索功能
- 支持导出元数据
- 性能进一步优化
